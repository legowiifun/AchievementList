# JSON Editing/Creation
- Can select a JSON
- Select format - save, individual game, or general games
- Brings up editing menu with various fields
- Fields are automatically filled in if JSON selected, and is parsed correctly
- Save button to save JSON
- If you did not automatically select JSON, can select save location
- Use datetime input for date/time fields
- Use file selection input for image and json file links
- Platform selections is through multi-select input box, with other having universal input box
